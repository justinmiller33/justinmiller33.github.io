Mentorship (and Internal) Integrations

Email, SMS, and Slack Communications wrappers looped into the Data Pipeline to detail progress updates and new information to both event leaders and internal staff.

Mailer + Texter in Pipeline

> Wrappers for traditional SMTP Email Sendouts (Mailer) and SMS Sendouts integrated with Mentorship through the EZTexting API
> Three main message triggers for event leaders: Event Commitment (intro email), Logistics Finalization (tree goal & species confirmation) and Event Page Finalization.
> Other scheduled message triggers providing reminders to event leaders prompting consistent involvement (e.g., encouraging them to recruit sponsors if they haven't in the first month of fundraising).
> Main content is contained in the email to all event leaders, a reminder to check spam inboxes and reply to the email (if neccessary) is in the text to the main event organizer.




Internal Pings & Warnings

> To give internal updates the Slack API Wrapper is used.
> This includes pings to mentorship when school data requires a second confirmation as well as pings to developers when an issue is detected in the Pipeline or an API Wrapper.
